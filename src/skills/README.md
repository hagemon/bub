# Bub Skills Directory

This directory contains Bub's built-in skills.

Each built-in skill is a standard Agent Skill directory: a `SKILL.md` file plus any supporting `scripts/`, `references/`, `assets/`, or other resources it needs. Bub follows the open format documented at agentskills.io and adds a few recommended authoring patterns in its docs.

Built-in skills are discovered from `src/skills` and can be overridden by workspace or user-level skills using the normal precedence rules.
